También aprendí a desarrollar aplicaciones publicadas en la web gracias a la libreria Streamlit de python lo cual evidencio en el proyecto del sprint 7. Me visualiso aportando Valor a través de los aprendizajes adquiridos con la realización d elos diferentes proyectos de analítica de información mediante las herramientas más demandadas en el sector.
